Program File Name: rab378_HW1.html
Program Description: This program incorporates a method to
                     calculate the Fibonacci sequence given
                     a user's input.
How to Run: Open HTML file in an internet browser(Chrome, Firefox, etc).
            Once opened, there will be a brief description of Fibonacci Numbers.
            To Compute fibonacci numbers enter the value in entry box next to "Enter Number:"
            and press the compute button or enter on keyboard. The output will be a table of
            Fibonacci numbers from 0 up-to and including the value entered in the entry box.
            The Fibonacci number for the value enter is also displayed in BOLD above the table,
            for easy retrieval. Warnings will appear if invalid or out of range values are entered
            in entry box.
